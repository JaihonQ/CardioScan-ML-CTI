# CardioScan-ML-CTI
Cardiovascular Disease Prediction System using Neural Network – Version 1 | CTI Academy Volunteer Instructor Project

---

## Overview
**CardioScan** is a cardiovascular disease prediction system built using a **Neural Network model**.  
This project demonstrates the development, training, and deployment of an end-to-end machine learning system focused on real-world cardiovascular data.

**Key Highlights:**
- End-to-end ML pipeline: preprocessing → model training → deployment  
- Demonstrates best practices in Neural Network development  
- Fully reproducible with **Python 3.12.3**  

---

## Project Structure
drawio/architecture.drawio      # System architecture diagram notebooks/model_ensemble.ipynb  # Neural Network model (Video 19) python/                         # Python scripts for CardioScan system data/                           # Sample cardiovascular dataset for demonstration
> Only `model_ensemble.ipynb` (Video 19) is required to train the Neural Network model.  
> Python scripts in the `python/` folder handle model deployment and system execution.

## Requirements
- Python **3.12.3**  
- Install required packages:
```bash
pip install -r requirements.txt
Usage Instructions
Ensure all folders (notebooks/, python/, data/) are in the same directory.
Open Jupyter Notebook model_ensemble.ipynb and run all cells to train the Neural Network model.
Execute Python scripts in the python/ folder to run the CardioScan system:
python main.py
The system will load the trained model and make predictions on the provided cardiovascular dataset.
